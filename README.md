# bananacss

> A brazilian css preprocessor

install your CLI locally:

```sh
$ [sudo] npm link
```

Run:

```sh
$ bananacss
```

```sh
$ bananacss -x
```

```sh
$ bananacss -y
```

Run unit tests with mocha:

```sh
$ npm test
```

Publish:

```sh
$ npm punlish
```
